Welcome to Morea

Easily generate educational websites organized by Modules, Outcomes, Readings, Experiences, and Assessments.

Make the pedagogy visible

Teachers often organize courses into a sequence of a dozen or more modules. Each module has one or more learning outcomes, which identify what the students should know if they successfully complete the module. The module's educational content often combines one or more passive readings (providing background and reference material) with one or more active experiences (typically homework assignments, events, or in-class group work). Finally, the teacher creates one or more assessment that determine if the students have achieved the learning outcomes.

Morea makes this (often implicit) course structure explicit through a set of pages that present the course content organized not just by modules, but also by learning outcomes, readings, learning experiences, and assessments. This helps students not only as they work on the current module, but when they review the content later (even after the course is over!)

Simplify development with Github

GitHub provides marvelous technology for design, development, and delivery of educational websites:

  • Free storage of content.
  • Free private repositories for educators.
  • Free hosting of course websites.
  • The ability to clone, branch, and fork repositories.

Morea leverages these capabilities to make it easier to create courses and improve them over time. Morea and Github help you collaborate with students and teachers to create and improve educational content.

Simplify presentation with Jekyll

Nice websites involve lots of CSS and HTML, which is a pain to maintain manually. Jekyll is a "static site generator" that enables you to develop your course content in Markdown format without using HTML.

Morea leverages Jekyll to separate HTML/CSS presentation details from your course content. Morea gives you a nice, responsive website without any presentation details in your content.
